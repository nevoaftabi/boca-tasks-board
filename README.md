# Boca Tasks Board

This is a task board application. A user can post tasks which they need help finishing. They will need to choose a title, description, category, and pay when submitting a task.

The task can be claimed by multiple people. Every person who claims a task will have to provide their name and email, and the ID of the task they are claiming.

* Node.js and Express.js
* TypeScript
* Malformed JSON error handling
* Data validation with Zod
* PostgreSQL database
* Prisma ORM
* Jest for unit testing
